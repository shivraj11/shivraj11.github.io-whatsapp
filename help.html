Hello, user

This is a simple messaging app, helps you to communicate with others around the globe, using internet in a very fast and secure way.

Create account : In the Setting dropdown box, with the help of "Create Account" link, fill/upload the mandatory details, finally click on "Save&Create" Button.

Update account : In the Setting dropdown box, with the help of "Edit profile" link, fill/upload the mandatory details, finally click on "Update Account" Button.

View Unread Chats : Under Chats section click on "view unread chats" button.


For more information click on "Learn more" button.
